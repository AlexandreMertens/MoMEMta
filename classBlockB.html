BlockB Class Reference

Detailed Description

Final (main) Block B, consisting on \(q_1 q_2 \to X + s_{12} \to X + p_1 p_2\), where \(q_1\) and \(q_2\) are Bjorken fractions, and \(s_{12}\) is a particle decaying into \(p_1\) (invisible particle) and \(p_2\) (visible particle).

This Block addresses the change of variables needed to pass from the standard phase-space parametrization to the \(\frac{1}{4\pi E_1} ds_{12} \times J\) parametrization.

The integration is performed over \(s_{12}\) with \(p_2\) as input. Per integration point, the LorentzVector of the invisible particle, \(p_1\), is computed based on the following set of equations:

  • \((p_1 + p_2)^2 = s_{12} = M_{1}^{2} + M_{2}^2 + 2 E_1 E_2 + 2 p_{1x}p_{2x} + 2p_{1y}p_{2y} + p_{1z}p_{2z}\)
  • Conservation of momentum (with \(\vec{p}_T\) the total transverse momentum of visible particles):
    • \(p_{1x} = - p_{Tx}\)
    • \(p_{1y} = - p_{Ty}\)
  • \(M_1 = 0 \to E_{1}^2 = p_{1x}^2 + p_{1y}^2 + p_{1z}^2\)

Up to two \(p_1\) solutions are possible.

  • Integration dimension: 0
  • Parameters:
    • energy (double): Collision energy.
  • Inputs:
    • s12 (double): Invariant mass of the particle decaying into the missing particle ( \(p_1\)) and the visible particle, \(p_2\). Typically coming from a BreitWignerGenerator module.
    • inputs (vector(LorentzVector)): LorentzVector of all the experimentally reconstructed particles. In this Block there is only one visible particle, \(p_2\).
  • Outputs:
    • invisibles (vector(vector(LorentzVector))): LorentzVector of the invisible particles. In this Block \(p_1\). One value per solution.
    • jacobians (vector(double)): Jacobian of the performed change of variables, leading to an integration on \(ds_{12}\) One jacobian per solution.
